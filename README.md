# LeetCode Solutions

This repository contains my solutions to various LeetCode problems.

## Purpose

- Keep my LeetCode workspace synchronized across multiple machines  
- Maintain a personal archive of solved problems  
- Track progress and revisit algorithms over time  

## Structure

The repository is flat and contains solution files directly at the root level.

Each file is named after the corresponding LeetCode problem ID:

- `112.py`
- `203.ts`
- `1.py`
- etc.

The file extension indicates the language used:
- `.py` → Python solutions  
- `.ts` → TypeScript solutions  

## Notes

- Code is written primarily for learning and practice  
- Some solutions may include multiple approaches or optimizations over time  
- This is a personal repository, not a curated interview guide  

## Sync Workflow

This repository acts as a centralized backup of my local LeetCode workspace across different environments.
